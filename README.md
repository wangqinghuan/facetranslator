# FaceTranslator

Real-time voice translation app for face-to-face communication between Chinese and English speakers.

## Features

- 🎤 Voice Recognition
- 🔄 Real-time Translation
- 🔄 Rotated Display - Chinese section rotated 180°, both parties see their own language upright

## Tech

- Web Speech API (Speech Recognition)
- MyMemory API (Translation)
- Pure HTML/CSS/JS
